## 🧨 Get Start

### Develop

저장소 클론하기

```bash
$ git clone https://github.com/withPet-team/withPet-client.git
```

패키지 설치 및 빌드하기

```bash
$ npm install
// 로컬 환경 실행
$ npm start
// 빌드 실행
$ npm run build
// 배포 진행
$ firebase deploy
```
